MyDotFiles
==========

This repository contains some important dotfiles that I use in my GNU/Linux systems.

As for now it contains:

- For bash:
	
	* Bashrc
	* Aliases
	* Functions.
	* Script for displaying git branch name in the prompt.

- For tmux:

	* tmux.conf
